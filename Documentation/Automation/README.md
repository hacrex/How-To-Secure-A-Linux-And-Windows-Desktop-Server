# Automation Scripts Documentation

This section provides detailed documentation and usage instructions for the Ansible playbooks and PowerShell scripts available in the `Automation/` directory. These guides aim to help users understand how to effectively deploy and manage the automated hardening steps for both Linux and Windows server environments.

## 📖 Table of Contents

- [Windows Automation with PowerShell](./Windows-Automation-Guide.md)

## Introduction

Automation is key to maintaining consistent security configurations across a fleet of servers. The scripts and playbooks provided in this repository are designed to streamline the process of applying security best practices, reducing manual effort and minimizing human error. This documentation will walk you through the prerequisites, usage, and customization options for each automation script.
